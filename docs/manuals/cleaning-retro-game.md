# レトロゲームクリーニング手順

## 1. クリーニングに必要なアイテム

### カセット用キット

デリケートな電子部品や光学面を扱うため、以下のアイテムを揃えてください。

- **無水エタノール：** 水分を含まないため、電子基板や端子の清掃に最適です（除菌用アルコールは水分を含むため避けてください）。
- **綿棒：** カセットの端子清掃用。
- **マイクロファイバークロス：** ディスク面の指紋や汚れを拭き取る用。
- **エアダスター：** 内部のホコリを飛ばす用。
- **メラミンスポンジ（限定利用）：** カセット外装の落ちにくい汚れ用（端子やディスクには厳禁）。

### 外装用キット

外装用としては以下を準備してください。

- **無水エタノール：** プラスチックケース（GC、MDのプラケース）用。
- **消しゴム（プラスチック消しゴム）：** **紙箱の黒ずみ**や、カセットへの鉛筆書き除去用。
- **セロハンテープ / マスキングテープ：** 紙箱についた細かいゴミや「値札シールの粘着残り」を除去する用。
- **シール剥がし液（雷神など）：** プラケース用（※紙箱には厳禁）。
- **乾いた綺麗な布（セーム革など）：** 仕上げの乾拭き用。

## 2. 部位別のクリーニング手順

### 紙箱（FC、GB、N64、GGなど）

紙箱は水分と溶剤が天敵です。基本は「ドライ」で攻めます。

1. **ホコリ除去：** 乾いた布で優しく表面を拭く。
2. **黒ずみ：** 軽い汚れや黒ずみは、**新品の消しゴム**で極めて軽くこすります。強くこすると印刷が剥げるので注意してください。
3. **値札剥がし：** ドライヤーで軽く温めて粘着剤を緩めてから、端からゆっくり剥がします。残ったベタつきは、セロハンテープの粘着面で「ペタペタ」と叩いて転写させて取ります（溶剤は紙に染みるので使いません）。

### プラケース（MD、GC、PCEなど）

プラスチックは比較的メンテナンスしやすいですが、変色に注意が必要です。

1. **表面の拭き上げ：** 無水エタノールを布に含ませ、表面の汚れや手垢を拭き取ります。
2. **シールのベタつき：** **シール剥がし液**が使えます。ただし、ジャケット（中の紙）を必ず抜いてから作業してください。
3. **スレ傷：** 深い傷はどうしようもありませんが、表面の曇りはプラスチッククリーナー（コンパウンド）で磨くと光沢が戻ります。

### カセット本体・カード（全般）

1. **端子：** 前述の通り、無水エタノール＋綿棒。
2. **外装の隙間：** ボタン周りや溝に溜まった汚れは、**爪楊枝や乾いた歯ブラシ**でかき出します。
3. **ラベル面：** ラベル（シール部）にエタノールをつけると、インクが滲んだり剥がれたりします。ここは固く絞った布で拭く程度に留めてください。

## 3. 「高く売る」ための仕上げ（技法36引用）

クリーニングが終わったら、以下の工夫をすることで商品価値が跳ね上がります。

- **OPP袋（クリスタルパック）への封入：**
  クリーニング直後に透明な袋に入れることで、「大切に保管されていた感」を演出できます。特に紙箱ソフトはこれだけで見た目が2ランク上がります。
- **端子の接点復活：**
  「接点復活剤を塗布済み」と記載するだけで、レトロゲーム特有の「動かないかも」という購入者の不安を払拭できます。
- **電池交換（GB/FCなど）：**
  もし可能であれば、バックアップ電池を新品に交換し「電池交換済み」と記載すれば、相場より数千円高く売れることも珍しくありません。

## 4. 各ハード別の特有注意点

| ハード           | 注意ポイント                                                                                                 |
| :--------------- | :----------------------------------------------------------------------------------------------------------- |
| **PCエンジン**   | Huカードの裏面（黒い部分）は傷が目立ちやすいので、眼鏡拭きのような柔らかい布で一方向に拭くこと。             |
| **メガドライブ** | カセットの質感がザラついているため、綿棒の繊維が残りやすい。最後にエアダスターで飛ばす。                     |
| **ゲームギア**   | 液晶画面（本体）の掃除もセットで行う場合、プラスチックパネルをアルコールで拭くと白濁する恐れがあるため注意。 |
