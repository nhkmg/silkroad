# ハピネット仕入れ手順

FBAやYahooショッピング売りを前提として、ハピネット(Happinet)で仕入れる手順を以下にまとめます。

---

## 前提条件

- 採用フルフィルメントサービス
  - ヤマトのフルフィルメントサービスは2025年2月に終了したので利用不可
  - そのため、このほかの候補となるフルフィルメントサービスは以下の通り
  - 楽天スーパーロジスティクス（RSL）
  - フィルフィルメント by Amazon（FBA）
  - [佐川グローバルロジスティクス（SGL）](https://www.sagawa-logi.com/)
  - [いつロジ](https://itsumo365.co.jp/service/fulfillment/itsulogi.html)
  - [ロジモプロ](https://www.logimopro.jp/y-fulfilment-campaign)
  - [ギークプラス](https://www.geekplus.jp/service/fulfillment/)
  - [STOCKCREW](https://stockcrew.co.jp/)
    - Yahoo!ショッピングや楽天をはじめ、Shopify、BASE、ecforceなど、主要モールやカートとAPI連携可能
    - 発送業務の「完全な自動化」を実現できる
    - ネクストエンジンを連携することで以下の自動化まで可能となる
      - Yahooショッピングへの注文ステータス完了
      - Yahooショッピングからの出荷通知メール
- 手数料は10%計算とする
  - 販売手数料: 約5.5% + プロパケ + PRオプション考慮
  - 配送料はヤマトFBA
- ハピネットの会員登録
  - B2B卸売会員として登録が必要
  - 法人格が必要（個人事業主の場合は事業開始届を提出）

## ハピネットの特徴

- おもちゃ・ホビー・ゲーム・アニメ関連商品が豊富
- 通常、小売価格より大幅に安い卸売価格を設定
- ボリュームディスカウント（数量購入で割引）あり
- 配送方法が複数あり
  - 当日配送（首都圏のみ）
  - 翌営業日配送
  - 定期配送（定期注文で割引）
- 返品制度あり（商品到着後7日以内）

## 手順概要

1. [ハピネット公式サイト](https://www.happinet.co.jp/)にアクセス
   - B2B向けの卸売サイト
2. ユーザーログインして商品を検索
   - カテゴリー別に絞り込み（おもちゃ、ゲーム、アニメグッズ等）
3. 人気順・売上順でソートして売れ筋商品を確認
4. Yahooショッピングで同一商品を検索して価格差を確認
5. 利益が取れる商品を特定
6. 最小ロット数（通常1個から購入可能）を確認
7. 在庫確認後、購入

## 仕入れ時の注意点

### 価格検証

- ハピネット価格 + 送料 + 手数料10% < 原価として計算
- Yahooショッピングでの平均販売価格を確認
- 商品ランキングで需要を確認

### 在庫管理

- 売上順でリピート商品を特定
- セール・キャンペーン商品は価格変動がある可能性
- 返品制度を活用してリスク軽減

### 配送方法の選択

- 当日配送：急ぎの場合だが割高
- 翌営業日配送：通常はこちら
- 定期配送：継続商品は割引率が高い

## トラブルシューティング

### 商品が見つからない場合

1. 検索キーワードを変更
2. カテゴリーから絞り込み
3. ハピネット営業担当に相談

### 価格が合わなくなった場合

- キャンペーン期間中は価格変動がある
- 定期的に価格をリセットして確認

### 配送遅延

- 営業日基準であることを確認
- 土日祝日は除外される
