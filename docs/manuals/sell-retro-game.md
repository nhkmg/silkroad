# レトロゲーム売却手順

**「レトロゲーム（500本＋本体）」を最速・最高値で現金化するための専用ロードマップ**を整理しました。

クリーニングが終わった「商品」を、いかに市場へプロっぽく提示して高値を引き出すかに特化しています。

## 1. 「証拠画像」の作成（TS3730のフル活用）

高額レトロゲームにおいて、買い手が最も恐れるのは「写真では見えないダメージ」です。スキャナーを使い、圧倒的な信頼感を演出します。

- **パッケージの「6面スキャン」**:
  [キヤノン TS3730](http://googleusercontent.com/shopping_content/0_link)で箱の正面・背面だけでなく、上下左右の側面（耳の折れや潰れがわかる部分）をスキャン。
  > **メリット**: スマホ撮影特有の「光の反射」や「歪み」がなく、状態がクリアに伝わるため、コレクターが安心して高値を入札できます。
- **端子の接写**:
  クリーニング済みのピカピカな端子をスマホのマクロモードで撮影。「通電の良さ」を視覚的にアピールします。
- **動作画面のキャプチャ**:
  「去年確認済み」の言葉だけでなく、実際にテレビに映っている画面を撮影して掲載します。

## 2. 「付属品」の完全リスト化

レトロゲームは「中身がどれだけ揃っているか」で価格が数倍変わります。

- **付属品チェック**: 箱、説明書に加え、ハガキ、チラシ、注意書き、当時の値札（あえて残すのも一興）、内袋の有無を確認。
- **完品（CIB）の明記**: すべて揃っているなら、商品説明の冒頭に**「【完品】付属品完備・コレクション級」**と記載します。

## 3. 法人流・信頼の商品説明テンプレート

「個人の不用品処分」ではなく、**「法人による厳格な管理品の放出」**という見せ方で差別化します。

> **【商品説明の構成例】**
>
> - **商品名**: （例：【美品】スーパー32X カオティクス 箱説ハガキ付 動作確認済）
> - **管理状況**: 「ゲーム開発研究用資産として暗所・非喫煙・防湿環境にて保管」
> - **状態詳細**: 「パッケージ角に微細なスレあり（画像3枚目参照）、端子清掃済み」
> - **発送**: 「JCB Biz ONE提携の配送サービスにて、厳重に緩衝材を使用し即日発送」

## 4. 販路の戦略的使い分け（改訂版）

500本＋大量のシールを、ストレスなく（メルカリを使わずに）効率よくさばくための出口戦略です。

| ランク          | 対象アイテム                           | 推奨販路       | 運用のポイント                                                                      |
| :-------------- | :------------------------------------- | :------------- | :---------------------------------------------------------------------------------- |
| **S級プレミア** | カオティクス、ダーナ、ブラックゼウス等 | **eBay**       | **世界市場へ。** 円安の恩恵を最大化。コレクター層が厚く、日本の数倍で売れることも。 |
| **A級〜中堅**   | VB本体、モンスターワールドIV、シール束 | **ヤフオク**   | **国内ガチ勢へ。** 「注目のオークション」設定で露出を上げ、終了時間を日曜夜に設定。 |
| **B級〜一般**   | 数千円以下のソフト、周辺機器、攻略本   | **Amazon FBA** | **完全自動化。** 在庫をAmazon倉庫へ送るだけ。顧客対応・発送を外注化し、時間を節約。 |

### 販路判定のフローチャート

基本的には以下の優先順位で振り分けます。

#### 1. eBay（海外輸出）：推定価格 3万円〜 ＋ 希少品

世界中のコレクターが相手なので、**「日本国内よりも海外で評価が高いもの」**を優先します。

- **判定基準**: 日本語が読めなくても遊べる（アクション・シューティング）、または日本限定発売のプレミア品。
- **対象**: 『カオティクス』『ダーナ』『バーチャルボーイ本体セット』など。
- **理由**: 円安の恩恵で国内相場の1.5〜2倍以上を狙える上、eBayのバイヤーは「プロ同士の取引」を好むため、法人出品との相性が抜群です。

#### 2. ヤフオク（国内競り）：推定価格 5,000円 〜 3万円 ＋ セット品

「海外に送るほどではないが、国内に熱心なファンがいる」層を狙います。

- **判定基準**: 付属品が完備されている「美品」、または単品では送料負けする「同一ハードのまとめ売り」。
- **対象**: PCエンジンの中堅タイトル、メガドライブの箱説付き、食玩シールのコンプセットなど。
- **理由**: オークション形式は「欲しい人たちが価格を吊り上げてくれる」ため、こちらが価格設定に悩む必要がありません。また、入札者の評価制限をかけることで、質の低いユーザーを排除できます。

#### 3. Amazon FBA（自動販売）：推定価格 〜 5,000円 ＋ バーコードあり

「1点ずつ出品・梱包するのが時間の無駄」と感じる一般ソフト用です。

- **判定基準**: バーコードがあり、Amazon内にカタログが存在するもの。
- **対象**: ファミコンやスーファミの一般タイトル、攻略本、周辺機器。
- **理由**: 推定価格が低くても、**「納品後はAmazonが勝手に売って発送もしてくれる」**という外注化メリットが大きいです。管理コストを時給換算すると、安価なものはここが正解です。

### 判定基準の早見表

| 項目               | **eBay**               | **ヤフオク**             | **Amazon FBA**           |
| :----------------- | :--------------------- | :----------------------- | :----------------------- |
| **主なターゲット** | 海外の富裕層・ガチ勢   | 国内の収集家・セット狙い | 一般ユーザー・即決層     |
| **推定売却価格**   | **高（3万円〜）**      | 中（5000円以上）         | 低〜中（5000円未満）     |
| **手間**           | 発送がやや面倒（海外） | 出品・梱包作業あり       | **ほぼゼロ（一括納品）** |
| **民度・トラブル** | 良好（ビジネスライク） | 普通（評価制限可能）     | 良好（Amazonが仲裁）     |

### 各販路の具体的な捌き方

#### ① eBay（海外プレミア・コレクター向け）

- **ターゲット**: 英語圏・欧州の富裕層コレクター。
- **メリット**: メルカリのような「民度の低い値下げ交渉」が少なく、状態の良さを評価するバイヤーが多い。
- **コツ**: TS3730でスキャンした高解像度画像を12枚フルに掲載。「Made in Japan」「Tested & Working」を強調します。

#### ② ヤフオク（セット売り・国内オークション）

- **ターゲット**: 国内のレトロゲーム愛好家、リセール業者。
- **メリット**: メルカリと異なり、オークション形式のため「競り」が発生し、予想以上の高値になる可能性があります。
- **コツ**: 1週間以内に撮影予定の「背表紙写真」をもとに、「MD名作50本セット」などのまとめ売りで一気に在庫を減らします。

#### ③ Amazon FBA（一般ソフト・効率重視）

- **ターゲット**: 手軽にすぐ遊びたい一般ユーザー。
- **メリット**: 商品をAmazonの倉庫に一括納品するため、1点ずつの梱包・発送作業から解放されます。
- **コツ**: バーコードのあるソフトや攻略本が対象。価格改定ツールを使えば、最安値に合わせて自動で売れていきます。

---

## 5. 「発送」をブランド化する

最後の手順ですが、ここを丁寧にすると「次の高額出品」への評価が爆上がりします。

- **クリスタルパック（OPP袋）**: 全ソフトをビニールに封入。
- **専用段ボール**: 100均などの使い回しではなく、Amazon等で買った新品の段ボールを使うと、受取時の評価（信頼度）が段違いです。
