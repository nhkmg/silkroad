# コジマ・ビックカメラ仕入れ手順

## 1. ドンキ×コジマのハシゴが有利な理由

- **カテゴリの補完関係**：
  ドンキは「日用品・消耗品」に強いですが、コジマは「型落ち家電」「PC周辺機器」「おもちゃ」に強く、美容系以外のジャンルを広げるのに最適です。
- **物理的な近さ**：
  新小平駅からドンキへ向かう途中に位置しており、駐車場も108台と広いため、車での大量仕入れにも適しています。
- **価格競争の回避**：
  ドンキの驚安商品はライバルが群がりやすいですが、家電量販店の「在庫処分品」は店舗ごとに異なるため、Amazonでの価格競争に巻き込まれにくい傾向があります。

## 2. コジマで狙うべき場所（美容系以外）

以下のエリアはFBAの審査に影響を与えず、利益を出しやすいポイントです。

- **「展示品・在庫処分」ワゴン**：
  レジ付近や通路にある「現品限り」の札がついたワゴンは必須です。マウス、キーボード、USBメモリなどのPC周辺機器は回転が良く、利益も取りやすいです。
- **季節家電の入れ替え期**：
  現在は春先（4月）ですので、冬物家電（加湿器、セラミックファンヒーター等）の最終処分や、これから需要が上がる扇風機の旧型番が狙い目です。
- **おもちゃ・ホビーコーナー**：
  コジマはビックカメラグループのため、おもちゃの在庫が豊富です。廃盤になったプラモデルや、セールになっている知育玩具はAmazonとの価格差が出やすいジャンルです。

## 3. コジマでの操作と支払い（JCB Biz ONE）

- **JCB Biz ONEの利用**：
  コジマでもJCB Biz ONEで1.0%の還元が受けられます。ただし、コジマのポイントカードを提示すると、カード払いの場合は還元率が2%下がる（通常10%→8%など）点に注意してください。
- **`https://www.google.com/search?q=%E3%83%93%E3%83%83%E3%82%AF%E3%82%AB%E3%83%A1%E3%83%A9.com`との比較**：
  店舗価格がAmazonより高くても、ビックカメラの公式通販サイト（`https://www.google.com/search?q=%E3%83%93%E3%83%83%E3%82%AF%E3%82%AB%E3%83%A1%E3%83%A9.com`）の方が安い場合があります。店員さんに「ネットと同じ価格になりませんか？」と交渉することで、利益商品に変わることもあります。

## 4. 注意点：美容家電の回避

コジマにはドライヤー、シェーバー、電動歯ブラシなどの「美容・健康家電」が大量にありますが、これらはAmazonのカテゴリで「ビューティー」や「ドラッグストア」に分類されるケースが多々あります。
**スマホアプリでスキャンした際、カテゴリが「家電・カメラ」や「ホーム＆キッチン」であることを必ず確認**してください。

## まとめ：小平仕入れルート案

1. **午前中：コジマ×ビックカメラ**で単価の高い家電やPC周辺機器をリサーチ（20時閉店のため先に行くのが無難）。
2. **夜間：ドン・キホーテ**で、コジマで見落とした日用品や、深夜帯の補充品をリサーチ（27時まで営業）。

このルートを組むことで、JCB Biz ONEの利用額も効率よく稼げ、美容系を避けた安全なアカウント運用が可能になります。
